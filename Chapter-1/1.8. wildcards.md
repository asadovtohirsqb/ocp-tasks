# Wildcards
Classes in the same package are often imported together. You can use a shortcut to `import` all the classes in a `package`.
```java
import java.util.*;    // imports java.util.Random among other things
public class NumberPicker {
  public static void main(String[] args) {
    Random r = new Random();
    System.out.println(r.nextInt(10));
  }
}
```
In this example, we imported _java.util.Random_ and a pile of other classes. The `*` is a wildcard that matches all classes in the package. Every class in the `java.util` package
