# O`zbekcha
## Klasslar

Java dasturlarida **klasslar (classes)** asosiy qurilish bloklari hisoblanadi.

Ko‘pgina klasslardan foydalanish uchun siz **obyektlar (objects)** yaratishingiz kerak bo‘ladi. Obyekt — bu klassning xotirada (memory) yaratilgan ishchi nusxasi, ya’ni dastur ishlayotgan vaqtdagi (runtime) ifodasidir. Shu sababli, obyekt ko‘pincha **klassning instansiyasi (instance)** deb ham ataladi, chunki u klassning bitta aniq ko‘rinishini ifodalaydi.

Turli klasslarga tegishli barcha obyektlar birgalikda dasturingizning holatini (state) ifodalaydi.

**Reference (murojaat)** esa — bu obyektga ishora qiluvchi o‘zgaruvchi bo‘lib, u orqali siz obyekt bilan ishlaysiz.

# English 
## Classes.
In Java programs, classes are the basic building blocks.
To use most classes, you have to create objects. An object is a runtime instance of a class in memory. An object is often referred to as an instance since it represents a single representation of the class.
All the various objects of all the different classes represent the state of your program. A reference is a variable that points to an object.
